# Requirements

## Blender 4.0

Download : https://download.blender.org/release/Blender4.0/

## Python 3.10

Download : https://www.python.org/downloads/release/python-3100/

# How to use

1. Download the repository:
    - Use git clone or download the project as a ZIP file and extract it.
2. Open the project
    - Open the .blend file with Blender 4.0.
3. Configure the script:
    - On the left side, you will see the main Python script.
    - Scroll all the way to the bottom of the code.
    - Edit the file path variable to point to your source file.
4. Execute
    - Click the Run Script (play icon) button.
5. Result
    - The extracted models will appear in the 3D viewport on the right side.
    - Images, audio, and other assets are automatically saved into a new folder (named after the source file) created in the same directory as your source file.
